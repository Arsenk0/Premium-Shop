# Contributing to Premium Shop 👟✨

We love contributions! Whether you're fixing a bug, adding a feature, or improving documentation, here's how you can help.

## 🚀 How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: `git checkout -b feature/your-feature-name`.
3. **Make Changes**: Implement your fix or feature.
4. **Test Your Changes**: Ensure everything works as expected.
5. **Commit & Push**: `git commit -m "Add some feature"` and `git push origin feature/your-feature-name`.
6. **Open a Pull Request**: Describe your changes and why they're needed.

## 📝 Code Standards

- Follow PEP 8 for Python code.
- Write clear, descriptive commit messages.
- Ensure all templates are properly localized using `{% trans %}`.

## 🛠️ Reporting Bugs

If you find a bug, please open an issue with:
- A clear title and description.
- Steps to reproduce the bug.
- Any relevant logs or screenshots.

Thank you for helping us make Premium Shop even better! ❤️
